# 소개

안녕하세요, **박민건**(MinGeon Park)입니다. 개발하며 배운 것들을 여기에 기록합니다.

요즘 관심사는 한 문장으로 요약됩니다. **엔진은 LLM, 제품은 규칙과 게이트.** LLM 에이전트에게 일을 맡기되, 규칙 문서와 스키마와 결정론적 검증으로 품질을 지키는 구조를 이것저것 만들고 있어요. 집의 헤드리스 Mac mini에 에이전트를 상주시켜 놓고 같이 일하는 중입니다.

이 블로그의 글도 같은 방식으로 만들어집니다. 초안은 에이전트와 함께 쓰고, 문체 규칙을 문서로 걸어두고, AI 티 정량 진단과 변경률 게이트를 통과해야 발행돼요.

## 만든 것들

- [mapcn-kr](https://github.com/DevMinGeonPark/mapcn-kr) — npx 한 줄로 설치하는 한국 행정구역 지도 (시도→시군구→행정동→리 드릴다운)
- [amux](https://github.com/DevMinGeonPark/amux) — 에이전트를 아는 tmux 세션 매니저
- [gaejo](https://github.com/DevMinGeonPark/gaejo) — 한국 연구 발표 개조식(箇條式) 변환·검증 도구
- [ai-stat-sheet](https://github.com/DevMinGeonPark/ai-stat-sheet) — GitHub 활동을 RPG 스탯 카드로
